# AI Resume Analyzer

AI Resume Analyzer is a web application that analyzes resumes and extracts technical skills from PDF files. It evaluates resumes and generates an ATS-style score based on detected skills.

## Features
- Upload resume in PDF format
- Extract text from resume
- Detect technical skills
- Generate resume score
- Simple web interface

## Tech Stack
- Python
- Flask
- HTML
- CSS
- PDFMiner

## How to Run

1. Clone the repository
git clone https://github.com/yourusername/AI-Resume-Analyzer.git

2. Go to the project folder
cd AI-Resume-Analyzer

3. Install dependencies
pip install flask pdfminer.six

4. Run the app
python app.py

5. Open in browser
http://127.0.0.1:5000
